# InterviewX AI

An AI-powered mock interview simulator designed for coding and system design preparation. Candidates can practice coding challenges, system design, and voice-based behavioral interviews with real-time feedback and scoring.

Live Site: https://interviewx-ai-one.vercel.app

## Features

- **Coding & System Design Simulator**: Write code in an interactive editor with test cases, or draw and answer system design questions with instant AI-driven reviews.
- **Voice Interviews**: Simulated real-time voice conversations for behavioral rounds with speech-to-text feedback.
- **User Dashboard**: Track previous coding sessions, view resume analysis reports, and manage interview schedules.
- **Upgrades & Monetization**: Built-in Stripe/Razorpay integration for premium membership upgrades and plan management.
- **Feedback Forms**: Candidate feedback forms to rate the platform directly from the dashboard.

## Tech Stack

- **Frontend**: Next.js 15 (App Router), Tailwind CSS, TypeScript
- **Backend**: FastAPI (Python), SQLAlchemy, PostgreSQL (Neon DB)
